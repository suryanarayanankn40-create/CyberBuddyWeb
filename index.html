How a phishing attack works

  1. Attacker creates a fake website or message.
  2. User receives an email, SMS or social media message.
  3. User clicks the link.
  4. User enters login or personal data.
  5. Attacker steals and misuses the data.

Real looking phishing examples

These images show how scammers copy real websites and messages to trick users.

Common phishing tricks

  • Urgent warnings like ‚ÄúYour account will be suspended‚Äù
  • Free gifts, rewards and lucky winner messages
  • Fake technical support calls or messages
  • Shortened or strange looking links
  • Spelling mistakes and unusual sender addresses

How to verify a suspicious link

  • Check the full website address carefully.
  • Do not trust only the logo or page design.
  • Search the website name in Google.
  • Scan links using trusted online scanners.

If you clicked a phishing link

  • Immediately change your password.
  • Log out from all devices.
  • Enable two-factor authentication.
  • Report the message as phishing.
  • Check your account activity.
